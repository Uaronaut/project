# Notes API Service

REST API сервис для управления заметками с поддержкой пользователей и тегов.

## Возможности
- CRUD операции с заметками
- Пагинация и фильтрация
- Пользовательская система
- Тегирование заметок
- Полное покрытие тестами

## Технологии
- Python 3.9
- Flask
- SQLAlchemy
- PostgreSQL
- Docker
- pytest
